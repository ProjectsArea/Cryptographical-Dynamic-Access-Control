Project Overview

CRYPT-DAC is an advanced cryptographic system implementing Distributed Access Control with secure file sharing capabilities using Shamir's Secret Sharing algorithm.

  • RSA Key Wrapping - Secure key distribution mechanisms
  • Shamir's Secret Sharing - Threshold-based access control
  • Role-Based Access - Granular permission management
  • AWS S3 Integration - Scalable cloud storage
